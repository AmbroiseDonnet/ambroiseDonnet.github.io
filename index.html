Projets

Projets Scolaires

Jeu Ludo-Pédagogique

Le jeu ludo-pédagogique est un projet dans lequel, par groupe de trois, nous avons créé un jeu destiné aux enfants de fin primaire et début collège pour qu'ils apprennent en jouant. Nous avions le choix du type de jeu à réaliser. Nous avons opté pour un RPG où il fallait battre des gardes en répondant à leurs questions pour progresser dans une carte générée à partir d'un fichier CSV.

Découverte de la méthode Agile

Nous avons eu une semaine sur le thème du framework Agile. Par groupe de cinq, nous avons conçu un jeu avec cette méthode. Nous avons créé *Infinite Dungeons*, un jeu dans lequel nous devions combattre des monstres au tour par tour et aller le plus loin possible.

Base de données Parcoursup

Nous avons élaboré et réfléchi à l'implémentation d'une base de données à partir d'un fichier CSV contenant des données issues de Parcoursup. Ensuite, nous avons établi un rapport statistique répondant à diverses questions pour lier ce projet au cours de statistiques.

Logiciel de classification

Nous avons réalisé un logiciel en JavaFX permettant de classifier un point à l'aide de l'algorithme KNN. Diverses fonctionnalités ont été implémentées : génération de graphiques, ajout de points, et choix des axes pour l'affichage des graphiques. Nous avons également relevé le défi de rendre le logiciel générique.
